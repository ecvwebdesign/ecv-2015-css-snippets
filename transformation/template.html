tutoriel

La propriété transformation va vous permettre de pivoter, augmenter, tourner, réduire ou incliner un élément en 2D ou bien 3D. Assez facile à mettre en oeuvre en 2D, les effets de transformations vous permettrons de vous distinguer dans certaines de vos présentations lorsqu’elles prennent une autre dimension en 3D ou bien combiner à l’effet de transition. À appliquer au bon endroit au bon moment, cette propriété peut s’avérer pratique dans certain cas.

Avant de commencer il est nécessaire de savoir écrire la bonne syntaxe:

					

						
css

						
						
transform: fonction(valeur);

Vous avez la possibilité de combiner les transformations en utilisant la syntaxe ainsi:

					

						
css

						
						
transform : function1(value1) function2(value2) function3(value3);

Pensez à utiliser les préfixes des navigateurs en question (-webkit-, -moz-, etc.) devant la propriété transform. Cela s'averera utile pour les utiliser sur les anciennes et nouvelles versions des navigateurs.

Let's start !!

1. Inclinaison alias Skew

Skew permet d’incliner (angle en degré) un élément sur l’axe x ou y ou les deux à la fois.

					

						
html


						
						
<div class="skeworiginal">Original</div>
<div class="skew">Incliner
sur X</div>

						
					


					

						
css


						
						
.skew
{
background-color: #5A9596;
-ms-transform: skewX(15deg); /* IE 9 */
-webkit-transform: skewX(15deg); /* Safari */
transform: skewX(15deg);
}

2. Échelle alias Scale

Scale augmente ou diminue la taille d’un élément sur son axe x ou y ou bien les deux à la fois. Dans l’exemple qui suit, l’élément div sera augmenté de 2 fois sa largeur d’origine et 3 fois sa hauteur d’origine.

					

						
html


						
						
<div class="scaleoriginal">Original</div>
<div class="scale">scale
sur X</div>

						
					


					

						
css


						
						
.scale{
background-color: #5A9596;
-ms-transform: scale(3,1.5); /* IE 9 */
-webkit-transform: scale(3,1.5); /* Safari */
transform: scale(3,1.5); /* Standard syntax */
width: 150px;
height: 100px;
}

3. Déplacement alias Translate

Translate permet de déplacer un élément de sa position actuelle (selon les paramètres indiqués pour x et y d’origine). L’exemple qui suit déplace l’élément div de 200 pixels vers la droite et 10 pixels vers le bas de sa position actuelle.

					

						
html


						
						
<div class="translate1">Original</div>
<div class="translate2">Translate</div>

						
					


					

						
css


						
						
.translate2{
background-color: #5A9596;
-ms-transform: translate(200px,10px); /* IE 9 */
-webkit-transform: translate(200px,10px); /* Safari */
transform: translate(200px,10px);
width: 250px;
height: 100px; 
}

4. Rotation alias Rotate

Rotate vous permet de pivoter un élément dans le sens horaire (ou anti horaire) selon un degré donné. L’exemple qui suit pivote l’élément div de 50 degré sur l'axe X. Il est aussi possible de paramétrer l'axe Y ou bien les deux.

					

						
html


						
						
<div class="rotationoriginal">Original</div>
<div class="Rotation 20°">Rotation 20°</div>

						
					


					

						
css


						
						
.rotation{
background-color: #5A9596;
width: 250px;
height: 100px;
-ms-transform: rotateX(50deg); /* IE 9 */
-webkit-transform: rotateX(50deg); /* Safari */
transform: rotateX(50deg);
}

5. Matrice alias Matrix

Matrix vous permet de combiner tous les transformations 2D en une. Les six paramètres enregistrés vous permette d’agrandir, pivoter, incliner, déplacer les éléments.

					

						
html


						
						
<div class="matrixoriginal">Original</div>
<div class="matrix">Transformé</div>

						
					


					

						
css


						
						
.matrix{
background-color: #5A9596;
width: 250px;
height: 100px;
-ms-transform: matrix(1, -0.3, 0, 1, 0, 0); /* IE 9 */
-webkit-transform: matrix(1, -0.3, 0, 1, 0, 0); /* Safari */
transform: matrix(1, -0.3, 2, 1, 0, 5);
}

6. Pour aller plus loin avec peu de 3D

En paramétrant un axe supplémentaire Z vous definissez un espace en 3D. Toutes les fonctions précédentes peuvent avoir une troisième valeur. L'exemple ci-dessous s'applique à la rotation sur Z.

					

						
html


						
						
<div class="rotationoriginal">Original</div>
<div class="rotation3D">Rotation sur Z</div>

						
					


					

						
css


						
						
.rotation3D{
background-color: #5A9596;
width: 250px;
height: 100px;
-webkit-transform: rotateZ(90deg); /* Safari */
transform: rotateZ(90deg);
}
