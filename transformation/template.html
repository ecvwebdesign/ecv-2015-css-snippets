tutoriel

La propriété transformation va vous permettre de pivoter, augmenter, tourner, réduire ou incliner un élément en 2D ou bien 3D. Assez facile à mettre en oeuvre en 2D, les effets de transformations vous permettrons de vous distinguer dans certaines de vos présentations lorsqu’elles prennent une autre dimension en 3D ou bien combiner à l’effet de transition. À appliquer au bon endroit au bon moment, cette propriété peut s’avérer pratique dans certain cas.

Avant de commencer il est nécessaire de savoir écrire la bonne syntaxe:

transform: fonction(valeur);

1. Inclinaison alias Skew

Skew permet d’incliner (angle en degré) un élément sur l’axe x ou y ou les deux à la fois.

					

						
html


						
						
<html>
	<head>
		<meta name="viewport" content="width=device-width, initial-scale=1">
		...
	</head>
	...
</html>
						
					


					

						
css


						
						
.txtRow {
	width: 64.0rem;
	margin: 0 auto;
}

.ttPrtTitle {
	text-transform: uppercase;
	margin: 0 0 2.0rem;
}

2. Échelle alias Scale

Scale augmente ou diminue la taille d’un élément sur son axe x ou y ou bien les deux à la fois. Dans l’exemple qui suit, l’élément div sera augmenté de 2 fois sa largeur d’origine et 3 fois sa hauteur d’origine.

					

						
css


						
						
@media screen and (min-width: 900px) {
	body {
		color: red;
	}
}
						

					

				
					

						
css


						
						
@media screen and (min-width: 500px) and (max-width: 900px) {
	body {
		background-color: black;
	}
}

3. Déplacement alias Translate

Translate permet de déplacer un élément de sa position actuelle (selon les paramètres indiqués pour x et y d’origine). L’exemple qui suit déplace l’élément div de 50 pixels vers la droite et 100 pixels vers le bas de sa position actuelle.

  • @media screen all : toutes résolutions
  • @media screen and (min-width: Xpx) : écrans de largeur minimale de X pixels
  • @media screen and (max-width: Xpx) : écrans de largeur maximale de X pixels
  • @media screen and (min-width: Xpx) and (max-width: Ypx) : écrans de largeur minimale de X pixels et de largeur maximale de Y pixels

Il existe d’autres conditions utilisables dans des cas plus rares, les lignes citées ci-dessus étant les plus couramment utilisées pour le responsive web design.

4. Rotation alias Rotate

Rotate vous permet de pivoter un élément dans le sens horaire (ou anti horaire) selon un degré donné. L’exemple qui suit pivote l’élément div de 20 degré vers la droite.

					

						
html


						
						
<link rel="stylesheet" media="screen and (min-width: 500px) and (max-width: 900px)" href="style1.css" />

5. Matrice alias Matrix

Matrix vous permet de combiner tous les transformations 2D en une. Les six paramètres enregistrés vous permette d’agrandir, pivoter, incliner, déplacer les éléments.

					

						
html


						
						
<picture>
	<source srcset="image-1.jpg" media="(min-width: 1000px)">
	<source srcset="image-2.jpg" media="(min-width: 500px)">
	<img src="image.jpg" alt="description de l’image">
</picture>

On n’oublie évidemment pas de rajouter la balise viewport entre les balises <head></head> de notre page HTML afin d’assurer le fonctionnement des media queries.
