Grid Layouts (ou "positionnement en grille") est une méthode de disposition. Elle permet de diviser l'espace des principales régions d'une page web en fonction d'une quantité fixe, de l'espace disponible dans la fenêtre du navigateur, ou d'une combinaison des deux. Pour résumé, la méthode Grid Layout permet une découpe en ligne et en colonne de votre interface comme nous le ferions pour un tableau de mise en page.

1. syntaxe

Pour ce familiariser rapidement et facilement avec Grid Layouts, il est important de comprendre les éléments qui composent cette grille ainsi que leurs terminologies.

  • Grid Lines : Les "Grid Lines" sont les lignes qui composent la grille. Elle peuvent être horizontales ou verticales. Nous pouvons référer à elles par des noms ou des numéros.
  • Grid Track : Une "Grid Track" correspond à l'espace entre deux lignes, horizontales ou verticales.
  • Grid Cell : Une "Grid Cell" correspond à l'espace entre quatre lignes. C'est l'unité la plus petite sur notre grille pour placer texte et/ou image.
  • Grid Area : Une "Grid Area" correspond à n'importe quelle zone sur la grille attachée par quatre lignes. Elle peu contenir un certain nombre de cellule (Grid Cell).

2. propriétés usuelles

Maintenant que vous en savez plus sur votre grille, nous allons voir comment interpréter ces terminologies en langage web.

  • ms-grid-column : Obtenir ou mettre une valeur qui spécifie dans quelle colonne de la grille l'on place l'objet/texte.
  • ms-grid-column-align : Obtenir ou mettre une valeur qui spécifie l'alignement horizontal de l'objet/texte dans la colonne de grille.
  • ms-grid-columns : Obtenir ou mettre une ou plusieurs valeurs qui spécifient la largeur de chaque colonne de grille.
  • ms-grid-column span : Obtenir ou mettre une valeur qui spécifie le nombre de colonnes de la griller que l'objet/texte enjambe.
  • ms-grid-row : Obtenir ou mettre une valeur qui spécifie dans laquel des rangés de la grille l'objet/texte est placer.
  • ms-grid-row-align : Obtenir ou mettre une valeur qui spécifie l'alignement vertical de l'objet/texte dans la rangée de la grille.
  • ms-grid-rows : Obtenir ou mettre une valeur qui spécifie la hauteur de chaque rangées e la grille.
  • ms-grid-row-span : Obtenir ou mettre une valeur qui spécifie le nombre de rangées de la grille que l'objet/texte enjambe.
  • ms-grid-cell : Positionnement d'un élément au sein d'une cellule nommé.
  • ms-grid-layer : Niveau du plan pour éléments superposés.

3. définir une grille

Nous commençons par écrire le code HTML simple de manière à obtenir la structure de base de la page mais aussi pour créer différentes "box" qui accueillerons les éléments de notre design :

					

						
html


						
						
...
<div class="Wrapper">
	<div class="box a">A</div>
	<div class="box b">B</div>
	<div class="box c">C</div>
	<div class="box d">D</div>
	<div class="box e">E</div>
	<div class="box f">F</div>
</div>

Nous allons maintenant créer dans le CSS l'élément Grilles puis spécifier les lignes et les colonnes. La première chose à faire est d'ajouter l'élément "display" et de lui attribuer la valeur ms-grid. Cette action crée l'élément grille. Dans l'élément grille, nous utilisons la propriété ms-grid-column pour spécifier la largeur de chaque colonne dans une liste délimitée par des espaces. Nous utiliserons aussi la propriété ms-grid-rows pour spécifier la hauteur de chaque colonne dans une liste délimité par des espaces :

					

						
html


						
						
...
<div class="Wrapper">
	<div class="box a">A</div>
	<div class="box b">B</div>
	<div class="box c">C</div>
	<div class="box d">D</div>
	<div class="box e">E</div>
	<div class="box f">F</div>
 </div>
						
					


					

						
css


						
						
.Wrapper{
	display: ms-grid;
	ms-grid-columns : 100px 10px 100px 10px 100px;
	ms-grid-rows: auto 10px auto;
}

4. le placement de base

Utiliser la grille déjà définie et positionner les éléments en subdivision afin d'organiser des "espaces de rangements".

					

						
css


						
						
.a { 
	grid-column-start: 1;
	grid-column-end: 2; 
	grid-row-start: 1;
	grid-row-end: 2;
}
.b {
	grid-column-start: 3;
	grid-column-end: 4; 
	grid-row-start: 1;
	grid-row-end: 2; 
}
.c { 
	grid-column-start: 5;
	grid-column-end: 6; 
	grid-row-start: 1;
	grid-row-end: 2;
}
.d { 
	grid-column-start: 1;
	grid-column-end: 2; 
	grid-row-start: 3;
	grid-row-end: 4;
}
.e { 
	grid-column-start: 3;
	grid-column-end: 4; 
	grid-row-start: 3;
	grid-row-end: 4;
}
.f { 
	grid-column-start: 5;
	grid-column-end: 6; 
	grid-row-start: 3;
	grid-row-end: 4;
}

Vous pouvez obtenir le même résultat en utilisant une syntaxe plus courte qui déclare la valeur de fin et la valeur de début. Les valeurs sont séparer par / :

					

						
css


						
						
.a { 
	grid-column: 1 / 2; 
	grid-row: 1 / 2;
}
.b { 
	grid-column: 3 / 4; 
	grid-row: 1 / 2;
}
.c { 
	grid-column: 5 / 6; 
	grid-row: 1 / 2;
}
.d { 
	grid-column: 1 / 2; 
	grid-row: 3 / 4;
}
.e { 
	grid-column: 3 / 4; 
	grid-row: 3 / 4;
}
.f { 
	grid-column: 5 / 6; 
	grid-row: 3 / 4;
}

Vous pouvez créer des espaces plus grand qu'une cellule individuelle en établissant une ligne de fin plus grande que celle-ci :

					

						
css


						
						
.a { 
	grid-column: 1 / 4; 
	grid-row: 1 / 2;
}
.b { 
	grid-column: 5 / 6; 
	grid-row: 1 / 4;
}
.c { 
	grid-column: 1 / 2; 
	grid-row: 3 / 4;
}
.d { 
	grid-column: 3 / 4; 
	grid-row: 3 / 4;
}

5. l'organisation de zone

Vous pouvez nommez les zones de votre grille afin d'y insérer du contenu spécifique. Pour ce faire, nous utiliserons la propriété grid-area. Répéter le nom d'une zone indique que celle-ci occupe un espace plus important sur la grille. Utilisé un . indique un espace vide.

					

						
html


						
						
...
<div class="Wrapper">
	<div class="box header">Header</div>
	<div class="box sidebar">Sidebar</div>
	<div class="box content">Content</div>
</div>
						
					


					

						
css


						
						
.sidebar {
	grid-area: sidebar;
}
.content {
	grid-area: content;
}
.header {
	grid-area: header;
}
.wrapper {
	display: grid;
	grid-template-columns: 100px 10px 100px 10px 100px;
	grid-template-rows: auto 10px auto;
	grid-template-areas: "header header header header header" 
	"sidebar . content content content";
}

6. le positionnement

L'outil Grid Layouts permet d'aligner les contenus verticalement ou horizontalement à l'aide des propriétés grid-row-align et grid-column-align dont les valeurs peuvent être les suivantes :

  • start : aligne l'élément au début de la cellule (gauche ou droite selon le sens de la lecture).
  • end : aligne l'élément à la fin de la cellule (gauche ou droite selon le sens de la lecture).
  • center : place l'élément au centre de la cellule.
  • stretch : étire l'élément (ses marges) pour occuper tout l'espace dans la cellule.
					

						
html


						
						
...
	<div class="Body">
		<div class="box navigation">Navigation</div>
		<div class="box galerie">Galerie</div>
		<div class="box article">Article</div>
		<div class="box info">Info</div>
	</div>
						
					


					

						
css


						
						
.body {
	 display: grid;
	 grid-columns: 250px 250px;
	 grid-rows: 250px 250px;
}
.navigation {
	 grid-column: 1; grid-row: 1;
}
.galerie {
	 grid-column: 2; grid-row: 1;
	 grid-row-align : center;
}
.article {
	 grid-column: 1; grid-row: 2;
	 grid-column-align : center;
}
.info {
	 grid-column: 2; grid-row: 2;
	 grid-row-align : center;
	 grid-column-align : center;
}

7. les unités de tailles

Le modèle d'affichage en grille ne se contente pas d'employer les unités de largeur et hauteur habituelles; il introduit des fonctions complexes permettant de s'adapter à des contextes de contenus différents :

  • px, %, em, ex, rem,… : pixels, pourcentages, em,… (unités courantes).
  • fr : raction(s) de l'espace restant.
  • min-content : se rapporte à la largeur (ou hauteur) de l'élément le plus petit.
  • max-content : se rapporte à la largeur (ou hauteur) de l'élément le plus grand.
  • minmax(min, max) : exemple minmax(min-content, 20%) correspond à largeur 20% (ou hauteur), mais au minimum largeur (ou hauteur) du contenu.
  • auto : s'adapte à la largeur (ou hauteur) du contenu.
  • fit-content : identique à auto et aussi à minmax (min-content, max-content).
					

						
css


						
						
.body {
	display: grid;
	grid-columns: 250px 1fr; /* largeur 250px et "espace restant" */
	grid-rows: 100px 1fr;
}
.navigation {
	grid-column: 1; grid-row: 1;
}
.galerie {
	grid-column: 2; grid-row: 1;
}
.article {
	grid-column: 1; grid-row: 2;
}
.info {
	grid-column: 2; grid-row: 2;
}

8. UNE GRILLE POUR TOUS TYPES D'ÉCRANS

L'un des avantages les plus flagrants d'une grille virtuelle, indépendante des éléments HTML qui peuvent s'y placer, est qu'il est extrêmement aisé de modifier la grille selon le contexte. Par exemple, pour s'adapter immédiatement aux différentes tailles d'écran, il suffit de... changer la grille initiale.

Associé aux CSS3 Media Queries, le modèle de positionnement en grille donne toute sa puissance, plus particulièrement si vous avez opté pour une trame sous forme de variante "template". En effet, seul le patron de départ nécessite d'être modifié : inutile d'intervenir sur la structure HTML, ni même sur le positionnement de chaque élément de page.

					

						
css


						
						
.body {
	display: grid;
	grid-template: "abcd";
}

@media (max-width: 640px) {
	 body {
		display: grid;
		grid-template: "a"
						"b"
						"c"
						"d";
	 }
}
