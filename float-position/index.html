Positioning & Float

Les propriétés de positionnement CSS vous permettent de positionner un élément librement dans votre page web.

  1. Le positionnement static
  2. Le positionnement absolu
  3. Le positionnement fix
  4. Le positionnement relatif
  5. La propriété z-index
  6. la propriété float

tutoriel

Les propriétés de positionnement CSS vous permettent de positionner un élément. Il peut également placer un élément derrière un autre, et de spécifier ce qui doit se produire lorsque le contenu d’un élément est trop long. Nous allons apprendre à modifier la position des éléments sur notre page. Vous allez voir, il existe plusieurs techniques permettant d’effectuer la mise en page du site. Chacune a ses avantages et ses défauts, ce sera à vous de sélectionner celle qui vous semble la meilleure selon votre cas.

1. Le positionnement static.

Les éléments peuvent être positionnés en utilisant le ‘top’, le ‘bottom’, le ‘left’, et le ‘right’ propriétés. Cependant, ces propriétés ne fonctionnera que si la propriété position est réglé en premier. Ils travaillent également différemment selon le procédé de positionnement.

Il existe quatre méthodes de positionnement différents. On va d'abord parler le positionnement 'static' qui est le positionnement par défaut, il n'est pas affecté par le ‘top’, le ‘bottom’, le ‘left’, et le ‘right’ propriétés.

					

						
html


						
						
<html>
	...
	<body>
		<h1>Hello css snippets!
		</h1>
	</body>
</html>
						
					


					

						
css


						
						
h1 {
	position: static;
}

2. Le positionnement absolu

Le positionnement absolu nous permet de placer un élément n’importe où sur la page (en haut à gauche, en bas à droite, tout au centre, etc.).

					

						
html


						
						
<html>
	...
	<body>
		<div class="rect">
		</div>
	</body>
</html>
						
					


					

						
css


						
						
.rect {
	position: absolute;
	bottom: 10px;
	left: 10px;
}

3. Le positionnement fix

l’élément fix reste toujours visible, même si on descend plus bas dans la page, il est positionné relativement par rapport à la fenêtre de navigateur.

					

						
html


						
						
<html>
	...
	<body>
		<div class="rect-fixed">
		</div>
	</body>
</html>
						
					


					

						
css


						
						
.rect-fixed{
	position: fixed;
	margin: auto;
	margin-top:40px;
}

4. Le positionnement relatif

Le positionnement relatif permet de décaler l’élément par rapport à sa position normale. Voici un example, le rectangle jaune est un élément relatif positionne par rapport un élément absolu (le rectangle bleu).

					

						
html


						
						
<html>
	...
	<body>
		<div class="content">
			<div class="rect-bleu">
				<div class="rect-jaune">
				</div>
			</div>
		</div>
	</body>
</html>
						
					


					

						
css


						
						
.content {
	width: 100%;
	height: 300px;
}

.rect-bleu {
	width:300px;
	height:200px;
	background-color: #79a9fd;
	position: absolute;
	margin-top: 40px;
	margin-left: 330px;
}

.rect-jaune{
	width:100px;
	height:100px;
	background-color: #fff186;
	position: relative;
	margin: 10px 0 0 20px;
}

5. La propriété z-index

La propriété z-index permet de placer un élément devant ou derrière les autres, elle ne marche pas avec le positionnement static.

					

						
html


						
						
<html>
	...
	<body>
		<div class="content">
			<img src="img/example.png" />
			<div class="rect-rose">
			</div>
			<div class="rect-jaune">
			</div>
		</div>
	</body>
</html>
						
					


					

						
css


						
						
img {
	width: 300px;
	height: auto;
	margin: 68px 0 0 25%;
	position: relative;
	z-index: 2;
}

.rect-rose {
	background-color: rgba(255,0,0,0.34);
	width:200px;
	height:200px;
	position: relative;
	margin: -150px 0 0 150px;
	z-index:1;
}

.rect-jaune {
	background-color: #f6ff7d;
	width:200px;
	height:200px;
	position: relative;
	margin: -290px 0 0 500px;
	z-index:3;
}

6. la propriété float

Il se trouve que cette propriété est aujourd’hui utilisée par la majorité des sites web pour faire la letterine de paragraphe ou faire la mise en page.

D’aboard nous l’avons utilisée pour faire flotter une letterine au début de la paragraphe. Utilisez F pour la letterine, puis changez le style css en utilisant span{}. On peut également utiliser une image flotte autour du texte.

					

						
html


						
						
<html>
	...
	<body>
		<div class="content">
			<p>				<span>F</span>loat left, The element floats to the left. Let the first letter "F" of a paragraph float to the left and style the letter.
			</p>
			
		</div>
	</body>
</html>
						
					


					

						
css


						
						
p {
	font-family: sans-serif;
	font-size: 16px;	
}

span {
	float: left;
	width: 0.7em;
	font-size: 55px;
	font-family: courier;
	line-height: 90%;
	color: #ce5a5a;
}

Utilisez le float pour faire la barre menu de votre site.

					

						
html


						
						
<html>
	...
	<body>
		<ul>
		
			<li class="hello">
			</li>
			<li class="menu">
			</li>
			<li class="menu">
			</li>
			<li class="menu">
			</li>
			
		</ul>
	</body>
</html>
						
					


					

						
css


						
						
ul {
	padding:0;
	width:80%;
	height:100px;
	margin-left:90px;
	list-style: none;
}

.menu {
	float:right;
}

.menu li {
	display: inline;
	padding:10px;
	margin-top: 55px;
}

.hello {
	float: left;
	font-size: 20px;
}

.menu a{
	text-decoration:none;
	font-size: 20px;
	background-color: #e2ff76;
	color:black;
	padding:5px 15px 5px 15px;
}

.menu a:hover{
	text-decoration:none;
	color:#e2ff76;
	background-color: black;
}
