What Are CSS3 Gradients?

CSS3 transitions are effects that let an element gradually change from one style to another.
To do this, you must specify two things:
• The CSS property you want to add an effect to.
• The duration of the effect.

With CSS3, we can add an effect when changing from one style to another, without using Flash or JavaScript.Mouse over the element below:

Exemple1

Position initiale de haut en bas :

* Note: This example does not work in Internet Explorer 9 and earlier versions.

Code:

.Degrade1 { background-image:linear-gradient(blue, red); padding: 100px;

Exemple2

position de gauche à droite

Code:

.Degrade2 { background-image:linear-gradient(to right,blue, red); padding: 100px; margin-bottom: 20px; }

Exemple3

Change position of the center:

Code:

.Degrade3 { background-image:linear-gradient(to right,blue 10px, red 90px); padding: 100px; margin-bottom: 20px; } }
Exemple4

modifie le degré du dégradé

Code:

.Degrade4 { background-image:linear-gradient(30deg,blue, red); padding: 100px; margin-bottom: 20px;
Exemple5

créer un dégradé radial.


Code:

.Degrade5 { background-image:radial-gradient(blue, red); padding: 100px; margin-bottom: 20px;
Exemple6

déplace le centre du radial.


Code:

.Degrade6 { background: radial-gradient(closest-side at 60% 55%,blue,red,yellow,white); padding: 100px; margin-bottom: 20px; }
