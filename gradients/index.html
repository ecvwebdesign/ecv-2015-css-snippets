Dégradés

Vous voulez de beaux dégradés sur votre site ? Très simple, avec ce tutoriel vous allez produire des dégradés de qualité.

tutoriel

Avant, pour faire des dégradés il fallait appliquer une image en background-image, mais maintenant de nouvelles propriétés permettent de réaliser et gérer des arrières-plans de teintes dégradées. Il s'agit de linear-gradient pour les dégradés linéaires et radial-gradient pour les dégradés radiaux.

1. La syntaxe pour linear-gradient.

Les dégradés linéaires empruntent la valeur-fonction linear-gradient() de la propriété background-image (ou background en raccourci). Il convient de préciser au minimum le point d'arrivée du dégradé ainsi que deux couleurs (ou plus).

				
					

						
css


						
						
.Degrade {

	background : linear-gradient(x y, couleur1, couleur2, couleurN)

	/* x et/ou y définissent la direction du dégradé. Exemple : "to right" ou "to left bottom",couleur : le nom, la valeur hexadécimale ou la valeur RGBa */

}

2. Créer un degradé de gauche à droite.

Lorsque l'on crée un dégradé on remarque que les deux couleurs se mélangent à 50/50 

					

						
html


						
						
<html>

	<...>

	<div class="Degrade2"> 	
	</div>

	<...>

</html>
						
					


					

						
css


						
						
.Degrade2 {

	background:linear-gradient(to right,blue ,red );

	/* spécifiez une hauteur à votre background car c'est un fond. */

	height:200px;
}

3. Un dégradé de haut en bas.

Changer le sens du dégradé est extrêmement simple, il suffit de mettre to bottom dans la syntaxte linear-gradient pour avoir un dégradé de haut en bas.

					

						
css


						
						
.Degrade2 {

	background:linear-gradient(to bottom,blue ,red );

	/* spécifiez une hauteur à votre background car c'est un fond. */

	height:200px;
}

4. Changer la position du centre du dégradé

On peut modifier le centre du dégradé en donnant les pourcentages de couleurs. 

					

						
css


						
						
.Degrade2 {

	background-image:linear-gradient(to right,blue 10%, red 90%);


	/* spécifiez une hauteur à votre background car c'est un fond. */

	height:200px;
}

Vous pouvez également créer plusieurs dégradés successifs et même choisir le pourcentage de chaques couleurs, comme ceci:

Il vous suffit de placer les couleurs les unes à la suite des autres 

					

						
css


						
						
.Degrade2 {

	background-image:linear-gradient(to right,blue,red,yellow,green);


	/* spécifiez une hauteur à votre background car c'est un fond. */

	height:200px;
}

5. Modifier le degré du dégradé

On peut carrement mettre un angle à notre dégradé pour un effet d'autant plus stylisé.

Il faut placer le degré choisis dans la direction

					

						
css


						
						
.Degrade2 {

	background-image:linear-gradient(30deg,blue, red);}

	/* spécifiez une hauteur à votre background car c'est un fond. */

	height:200px;
}

6. Dégradé de la couleur à la transparence

Pour encore plus de style tu peux créer de majestueux fondus, pour celà nous allons créer un dégradé de la couleur à la transparence. 

					

						
css


						
						
.Degrade2 {

	background-image:linear-gradient(to right,blue , rgba(255,255,255,0));

	/* spécifiez une hauteur à votre background car c'est un fond. */

	height:200px;
}

Pour se faire, nous avons agis sur la quatrième couche alpha que nous avons mis à 0 pour lui donner une totale transparence, mais vous pouvez modifiez le pourcentage de la transparence en réglant la couleur de 0 à 255

7. Syntaxe pour radial-gradient

Les dégradés radiaux sont réalisables à l'aide de la valeur-fonction radial-gradient(). Elle nécessite d'indiquer la forme du dégradé : circulaire (circle) ou elliptique (ellipse) ainsi que son point de départ.

				

					

						
css


						
						
.Degrade {

	background : radial-gradient(forme at x y, couleur1, ..., couleurN)

	/* x et/ou y définissent le point d'origine du dégradé radial, 
	couleur : le nom, la valeur hexadécimale ou la valeur RGBa */
}

8. Créer un dégradé radial

Pour le radial-gradient il faut choisir le centre et où le situé, dans le cas ci dessous nous avons placer le centre en haut et au centre du cadre

					

						
css


						
						
.Degrade2 {

	background: radial-gradient(ellipse at center top, red, blue);

	/* spécifiez une hauteur à votre background car c'est un fond. */

	height:200px;
}
