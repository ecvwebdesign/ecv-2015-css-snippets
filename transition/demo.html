Transition DEMO

Que-ce que c'est la transition dans CSS?

Avec CSS3, on peut ajouter les effets de changement de forme et animer les pages web uniquement à l'aide de styles CSS, sans apport de JavaScript.
Grâce aux dernières évolutions du langage et au module CSS3 Transitions, il est désormais possible de réaliser des transitions basiques à l'aide de CSS dans les navigateurs récents.

Passez votre souris sur les éléments en base:

couleur
déplace
alonger
fade
fade-out

On peut utiliser css transition avec transform pour avoir plus d'effet:

rotation
grandir
skew
bounce
tada
wobble

On peut utiliser css animation avec pour avoir plus d'effet:

flipInX
swing
shake
rubberBand
flash
SpeedIn
