1.Que-ce que c'est la transition dans CSS?

Grâce aux dernières évolutions du langage et au module CSS3 Transitions, il est désormais possible de réaliser des transitions basiques à l'aide de CSS dans les navigateurs récents.

2.Code basique de transition

  • transition: un raccourci pour fixer les quatre propriétés de transition dans une propriété unique.
  • transition-property: spécifie le nom de la propriété CSS qui a besoin de l'effet de transition.
  • transition-duration: spécifie le nombre de secondes ou milli-secondes un effet de transition prend pour terminer.
  • transition-timing-function: spécifie la courbe de vitesse de l'effet de transition.
  • transition-delay: spécifie un délai ( en secondes) pour l'effet de transition.
					

						
css


						
						
.div {
	background-color: #31464e;
	transition-property: background-color; 
	transition-duration: .5s;
	transition-timing-function: linear;
	transition-delay: 1s;
 }

La version abrégée：

					

						
css


						
						
.div {
	background-color: #31464e;
	-webkit-transition: background-color .5s linear, 1s; /* Safari */
	-o-transition: background-color .5s linear, 1s; /* Opera */
	-moz-transition: background-color .5s linear, 1s; /* Firefox */
}

Pour crée un effet de transition, on dois spécifier deux points:

  • le propriété de CSS qu’on veut ajouter les effets.
  • la duration des effets.

3. Transition timing-function

La propriété transition - timing-function peut avoir les valeurs suivantes:

  • ease - spécifie un effet de transition avec un démarrage lent, puis rapide, puis se arrête lentement ( ce est par défaut).
  • linear - spécifie un effet de transition à la même vitesse du début à la fin.
  • ease-in - spécifie un effet de transition avec un démarrage lent.
  • ease-out - spécifie un effet de transition avec une extrémité lente.
  • ease-in-out - spécifie un effet de transition avec un démarrage lent et à la fin.
  • cubic-bezier(n,n,n,n) - nous permet de définir nos propres valeurs dans une fonction cubique - Bézier.

4. Transitions en proportion data

					

						
html


						
						
<div class="data" id="data1">50%</div>
<div class="data" id="data2">80%</div>
<div class="data" id="data3">30%</div>
						
					

					

						
css


						
						
.data{
	background-color:#5A9596; 
	text-align: center;
	width: 50px;
	height: 50px;
	margin: 10px;
	padding: 5px;
	color: white;
	font-size: 20px;
	transition:width 2s ease,	background-color 2s ease;
}

.data:hover{
	background-color: #a88963;
}

#data1:hover{
	width:440px;
}
#data2:hover{
	width: 700px;
}
#data3:hover{
	width: 200px;
}

5. Transitions en surlignant menus

					

						
html


						
						
<div class="sidebar">
	<p><a class="menuButton" href="home">Home</a></p>
	<p><a class="menuButton" href="about">About</a></p>
	<p><a class="menuButton" href="contact">Contact Us</a><p>
	<p><a class="menuButton" href="links">Links</a></p>
</div>
						
					

					

						
css


						
						
.menuButton {
	display: block;
	text-align: center;
	background-color: #5A9596;
	width: 80%;
	height: 50px;
	color: white;
	font-family: sans-serif;
	font-size: 20px;
	text-decoration: none;
	padding: 6px;
	border: solid 3px white;
	margin: 0 auto;
	transition-property: background-color, color, border;
	transition-duration: .4s;
	transition-timing-function: ease-out;
	transition: all .5s ease-out;
}

.menuButton:hover {
	transition-property: background-color, color, border;
	transition-duration: .5s;
	transition-timing-function: ease-out;
	transform: scale(1.2);
	background-color:white;
	color:#5A9596;
	border: solid 3px #5A9596;
}

6. Transitions en image galerie

					

						
html


						
						
<div class="ext" ontouchstart=
"this.classList.toggle
('hover');">
	<div class="int">
		<div class="front">
		<!--votre contenu-->
		</div>

		<div class="back">
		<!--votre contenu-->
		</div>
	</div>
</div>

						
					

					

						
css


						
						
.ext { 
	perspective: 1000; 
}

.ext,
.front,
.back {
	height: 260px;
	width: 190px;
}

.ext:hover .int, 
.ext.hover .int { 
	transform: rotateY(180deg); 
}

.int { 
	transition: .75s;
	transform-style: preserve-3d;
}

.front, 
.back {
	-visibility: hidden;
	background: #FFF; 
	border: 1px solid #666; 
	position: absolute;
	top: 0;
	left: 0;
}

.front { 
	index: 2; 
}

.back { 
	transform: rotateY(180deg); 
}
