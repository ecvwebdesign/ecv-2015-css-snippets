1. LES TRANSITIONS, QUE-CE QUE C'EST ?

Grâce aux dernières évolutions du langage et au module CSS3 Transitions, il est désormais possible de réaliser des transitions basiques à l'aide de CSS dans les navigateurs récents.
Comme premier exemple, je vais vous montrer quelque chose de vraiment basique (je vous promets, la prochaine étape sera plus drôle !)
Passez votre souris sur les deux ronds ci-dessous :

2.BASES SUR TRANSITION

  • transition: un raccourci pour fixer les quatre propriétés de transition dans une propriété unique.
  • transition-property: spécifie le nom de la propriété CSS qui a besoin de l'effet de transition.
  • transition-duration: spécifie le nombre de secondes ou milli-secondes un effet de transition prend pour terminer.
  • transition-timing-function: spécifie la courbe de vitesse de l'effet de transition.
  • transition-delay: spécifie un délai (en secondes) pour l'effet de transition.
					

						
css


						
						
.div {
	background-color: #31464e;
	transition-property: background-color; 
	transition-duration: .5s;
	transition-timing-function: linear;
	transition-delay: 1s;
 }

La version abrégée “shorthand”：

					

						
css


						
						
.div {
	background-color: #31464e;
	-webkit-transition: background-color .5s linear, 1s; /* Safari */
	-o-transition: background-color .5s linear, 1s; /* Opera */
	-moz-transition: background-color .5s linear, 1s; /* Firefox */
}

Pour crée un effet de transition, on dois spécifier deux points:

  • •la propriété de CSS sur laquelle on veut ajouter des effets;
  • •la durée des effets.

L'effet de transition commencera lorsque la propriété CSS spécifiée (ici, couleur) change de valeur .
Maintenant, précisons une nouvelle valeur pour la propriété : width. Lorsqu'un utilisateur passe la souris sur l'élément…

3. TRANSITION TIMING-FUNCTION

La propriété transition - timing-function peut avoir les valeurs suivantes:

  • ease - spécifie un effet de transition avec un démarrage lent , puis rapide, puis s’arrête lentement (par défaut).
  • linear - spécifie un effet de transition à la même vitesse du début à la fin.
  • ease-in - spécifie un effet de transition avec un démarrage lent.
  • ease-out - spécifie un effet de transition avec une fin lente.
  • ease-in-out -spécifie un effet de transition avec un démarrage et une fin lents.
  • cubic-bezier(n,n,n,n) - nous permet de définir nos propres valeurs dans une fonction cubique de type courbes de Bézier.

4.EXEMPLE 1 : DATAVISUALISATION

Avec la transition, on peut faire des effets pour illustrer des proportions dans le cadre de datavisualisation, par exemple.
Tout d’abord, on dessine un cube avec width: 50px et background color: #5A9596. En suite, au passage de la souris, on change ses propriétés : width et background color. On l’écrit comme ça : -webkit-transition: width 2s ease, background-color 2s ease; /* Safari */ , la transition va durer 2s avec la vitesse ease. Sur l’état hover, on écrit les valeurs de changement. 

					

						
html


						
						
<div class="data" id="data1">50%</div>
<div class="data" id="data2">80%</div>
<div class="data" id="data3">30%</div>
						
					

					

						
css


						
						
.data{
	background-color:#5A9596; 
	text-align: center;
	width: 50px;
	height: 50px;
	margin: 10px;
	padding: 5px;
	color: white;
	font-size: 20px;
	transition:width 2s ease,	background-color 2s ease;
}

.data:hover{
	background-color: #a88963;
}

#data1:hover{
	width:440px;
}
#data2:hover{
	width: 700px;
}
#data3:hover{
	width: 200px;
}

5.EXEMPLE 2 : SURLIGNAGE DES LIENS

Avec la transition, on peut aussi ajouter des effets de mouvements pour les liens, dans un menu, par exemple.
On peut ajouter une propriété CSS transform pour avoir un effet plus intéressant. L’attribut transform: scale(1.2) est fait pour avoir un effet au passage de la souris, modifiant la taille de l’éléments pour qu’elle soit 1.2 fois plus grand que l’origine. 

					

						
html


						
						
<div class="sidebar">
	<p><a class="menuButton" href="home">Home</a></p>
	<p><a class="menuButton" href="about">About</a></p>
	<p><a class="menuButton" href="contact">Contact Us</a><p>
	<p><a class="menuButton" href="links">Links</a></p>
</div>
						
					

					

						
css


						
						
.menuButton {
  display: block;
  text-align: center;
  background-color: #5A9596;
  color: white;
  font-size: 20px;
  text-decoration: none;
  padding: 20px 50px 20px 50px;
  border: solid 3px white;
  -webkit-transition: background-color 1s, color 1s, border 1s, -webkit-transition 1s; /* Safari */
   -o-transition: background-color 1s, color 1s, border 1s, -webkit-transition 1s; /* Opera */
    -moz-transition: background-color 1s, color 1s, border 1s, -webkit-transition 1s; /* Firefox */
}

.menuButton:hover {
  
  -webkit-transform: scale(1.1);
  transform-origin: center center;
  background-color:#ffffff;
  color:#5A9596;
  border: solid 3px #5A9596;
	transition-property: background-color, color, border;
	transition-duration: .4s;
	transition-timing-function: ease-out;
	transition: all .5s ease-out;
}

6. EXEMPLE 3 : GALERIE D’IMAGES

Avec les transitions et la propriété transform, on peut aussi ajouter des effets sur les images. 

					

						
html


						
						
<div class="ext" ontouchstart=
"this.classList.toggle
('hover');">
	<div class="int">
		<div class="front">
		<!--votre contenu-->
		</div>

		<div class="back">
		<!--votre contenu-->
		</div>
	</div>
</div>

						
					

					

						
css


						
						
.ext { 
	perspective: 1000; 
}

.ext,
.front,
.back {
	height: 260px;
	width: 190px;
}

.ext:hover .int, 
.ext.hover .int { 
	transform: rotateY(180deg); 
}

.int { 
	transition: .75s;
	transform-style: preserve-3d;
}

.front, 
.back {
	-visibility: hidden;
	background: #FFF; 
	border: 1px solid #666; 
	position: absolute;
	top: 0;
	left: 0;
}

.front { 
	index: 2; 
}

.back { 
	transform: rotateY(180deg); 
}
