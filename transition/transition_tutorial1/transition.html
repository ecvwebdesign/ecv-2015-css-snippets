What Are CSS3 Transitions?

CSS3 transitions are effects that let an element gradually change from one style to another.
To do this, you must specify two things:
• The CSS property you want to add an effect to.
• The duration of the effect.

With CSS3, we can add an effect when changing from one style to another, without using Flash or JavaScript.Mouse over the element below:

Example1

Add a transition effect on the width property, with a duration of 2 seconds:

60%
100%

* Note: This example does not work in Internet Explorer 9 and earlier versions.

Code:

.div{
text-align: center;
padding: 10px;
color: #ffffff;
width: 100px;
height: 30px;
background: #9cb9b4;
-webkit-transition: width 2s; /* For Safari 3.1 to 6.0 */
-o-transition: width 2s; /* For Opera 10.5 to 12.1 */
-moz-transition: width 2s; /* For Firefox 4.0 to 16.0 */
transition: width 2s;
}

.div:hover{
width: 300px;
background-color: #88ab83;
}

Example4

Bigger the photo:

Code:

.div img{
width: 30%;
height: 30%;
margin-top: 50px;
margin-left: 10px;
margin-bottom: 50px;
display: block;
position: relative;
max-width: 100%;
height: auto;
}

.div img{
-webkit-transition: all .4s ease-out;/* For Safari 3.1 to 6.0 */
-o-transition: all .4s ease-out;/* For Opera 10.5 to 12.1 */
-moz-transition: all .4s ease-out;/* For Firefox 4.0 to 16.0 */
transition: all .4s ease-out;
}

.div img{
width: 30%;
height: 30%;
-webkit-transform: scale(1.2);/* For Safari 3.1 to 6.0 */
-o-transform: scale(1.2);/* For Opera 10.5 to 12.1 */
-moz-transform: scale(1.2);/* For Firefox 4.0 to 16.0 */
transform: scale(1.2);
}
Example5

The box below combines transitions for: width, height, background-color, transform. Hover over the box to see these properties animated.


Code:

.div{
border-style: solid #88ab83;
border-width: 2px;
display: block;
width: 100px;
height: 100px;
background-color: #4d6357;
border-radius: 10px;
-webkit-transition: width 2s, height 2s, background-color 2s, -webkit-transform 2s;/* For Safari 3.1 to 6.0 */
-o-transition: width 2s, height 2s, background-color 2s, -webkit-transform 2s;/* For Opera 10.5 to 12.1 */
-moz-transition: width 2s, height 2s, background-color 2s, -webkit-transform 2s;/* For Firefox 4.0 to 16.0 */
transition:width 2s, height 2s, background-color 2s, transform 2s;
}

.div:hover {
background-color: #9cb9b4;
width:200px;
height:200px;
-webkit-transform:rotate(180deg);/* For Safari 3.1 to 6.0 */
-o-transform:rotate(180deg);/* For Opera 10.5 to 12.1 */
-moz-transform:rotate(180deg);/* For Firefox 4.0 to 16.0 */
transform:rotate(180deg);
}
