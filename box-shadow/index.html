BOX SHADOW

La propriété de feuille de style css box-shadow permet d'ajouter un effet d'ombre à un élément HTML, comme du texte ou encore un bloc.

  1. Les valeurs des box-shadow

tutoriel

Les ombres portées prennent aujourd’hui une place importante dans le web design et permettent la mise en avant d’éléments importants, ou l’ajout d’une touche de profondeur dans vos graphismes. Il est évident, comme tout effet graphique, que l’utilisation des ombres doit être faite avec parcimonie et ne pas gêner l'utilisateur.
Deux types d’ombres existent en CSS, les box-shadow (les ombres ajoutées aux éléments de type bloc) et les text-shadow (les ombres ajoutées au contenu textuel). Nous allons nous occuper ici plus spécifiquement des box-shadow, qui fonctionnent avec plusieurs navigateurs comme Chrome 10, Safari 3.2, Firefox 3.6, SeaMonkey, MSIE 9+, Opera 10.6.

1. Les valeurs des box-shadow.

Dans la propriété de notre box-shadow, nous avons trois valeurs différentes exprimées en px (pixel) ainsi qu'une couleur hexadécimale. La feuille de style css box-shadow accepte comme valeur (dans l'ordre) :

  • inset: valeur optionnelle (mot clé), si elle est spécifier l'ombre est à l'intérieur sinon l'ombre est à l'extérieur de l'élément HTML.
  • x: valeur numérique obligatoire qui spécifie la distance horizontale de l'ombre, si elle est négative l'ombre est placée à gauche.
  • y: valeur numérique obligatoire qui spécifie la distance verticale de l'ombre, si elle est négative l'ombre est placée au dessus.
  • blur-radius: valeur optionnelle numérique positive suivie de px ou % ou pt ou em, correspond à la largeur de l'estompage de l'ombre (flou). Pour la valeur 0 l'ombre est nette.
  • color: valeur optionnelle elle indique la couleur de l'ombre. La couleur sélectionnée dépendra du navigateur elle est exprimée en hexadécimal (#000000) ou en couleur rgba.

ATTENTION il faut également rajouter dans le code css les différents préfixes des navigateurs afin que votre magnifique box-shadows soit visible par tout le monde!

Vous pouvez également spécifiez une ombre pour chaque coté. Il vous suffit de répéter 4 fois les valeurs de box-shadow séparer par une virgule. Vous pouvez aussi changer les autres valeurs. <div style="margin-left:20px;width:350px;height:80px;box-shadow:0 10px 5px 0 #000,10px 0 5px 0 blue, 0 -10px 5px 0 red ,-10px 0 5px 0 gold;">box-shadow:0 10px 5px 0 #000, 10px 0 5px 0 blue, 0 -10px 5px 0 red , -10px 0 5px 0 gold;>

					

						
html


						
						
<html>
	<head>
		<div class="shadow1">
		...
	</head>
	...
</html>
						
					


					

						
css


						
						
.shadow1 {
	background-color: white;
	height: 300px;
	margin-bottom: 50px;

	box-shadow: inset -5px -5px 5px #888;

	-moz-box-shadow: inset -5px -5px 5px #888;
	/* prefixe (-moz) pour firefox*/
	-webkit-box-shadow: inset -5px -5px 5px #888;
	/* prefixe (-webkit) pour chrome-opera-safari*/
	
	-ms-box-shadow: inset -5px -5px 5px #888;
	/* prefixe (-ms) pour internet explorer*/

	-o-box-shadow: inset -5px -5px 5px #888;
	/* prefixe (-o) pour opera*/
}

2. Et aussi les text-shadow!

La propriété text-shadow permet, comme son nom l'indique, de rajouter une ombre portée aux contenus textuels. La syntaxe est la suivante: text-shadow: h-pos v-pos (blur) (color);

Dans la propriété de notre text-shadow, nous avons quatres valeurs différentes. La feuille de style css text-shadow accepte comme valeur (dans l'ordre) :

  • h-pos : Position horizontale de l’ombre. Il s’agit de son décalage par rapport à la position de l’élément à ombrer sur l’axe des X. La valeur peut également être négative
  • v-pos : Position verticale de l’ombre. Il s’agit de son décalage par rapport à la position de l’élément à ombrer sur l’axe des Y. La valeur peut également être négativ
  • blur : Distance du flou, correspond à l’étendu du dégradé de la couleur opaque vers le transparent. (optionnelle - défaut : 0)
  • color : Couleur de l’ombre. Définit la couleur de votre ombre, voire son opacité grâce à la gestion de la couche Alpha. (optionnelle - défaut : valeur de la propriété color de l'élément.)

Comme pour box-shadow, les spécifications précisent que sans couleur renseignée pour l'ombre, la valeur de la propriété color de l'élément sert de référent à la couleur de l'ombre. Cependant, si Firefox et Opera attribuent la valeur de la propriété CSS color du texte, Chrome et Safari attribuent du transparent, la couleur semble donc obligatoire sur WebKit.

L'ombre portée utilisée simplement peut permettre d'ajouter un peu de profondeur au contenu textuel. C'est idéal pour un titre par exemple. Voici la syntaxe de l'ombre :

					

						
css


						
						
text-shadow: 1px 2px 3px rgba(0,0,0, 0.5);
